## 4. YouTube Shorts SEO Best Practices (2026)

### Algorithm Priorities
- **Viewed vs Swiped ratio**: Target 70%+ viewed rate
- **Replay value**: If viewers watch 1.5x, algorithm flags as "extraordinary value"
- **Completion rate**: 50s Short with 90% retention beats 15s with 60%
- **First hour engagement**: Likes/shares in first 60 minutes are critical

### Optimal Format
- **Length**: 30-55 seconds (higher loop rate than 7-15s clips)
- **Hook**: First 2 seconds must grab attention (question, surprising stat, bold claim)
- **Resolution**: 1080x1920 vertical
- **Captions**: Always on (85% of Shorts watched without sound)
- **CTA**: End with "Follow for more" or question to drive comments

### Title Formula
```
[Hook/Number] + [Keyword] + [Benefit/Surprise]
```
Examples:
- "I found 47 SEO issues in 30 seconds with AI"
- "Free backlink checker that nobody knows about"
- "This AI replaced my $3000/month SEO agency"

### Description Formula
```
Line 1: Primary keyword + context (most weight for search)
Line 2: Secondary keyword + value proposition
Line 3-5: Hashtags (3-5 max)
```

### Posting Strategy
- **Frequency**: 3-5 Shorts per week minimum
- **Best times**: Tue-Thu, 12pm-3pm and 7pm-9pm (test your audience)
- **Series batching**: Record 5-10 Shorts in one session, release over 2 weeks
- **Cross-post**: Same content on TikTok and Instagram Reels (different hashtags)

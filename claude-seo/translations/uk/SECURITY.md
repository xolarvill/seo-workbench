# Політика безпеки

## Повідомлення про вразливість

Якщо ви виявили вразливість безпеки, повідомте про неї відповідально:

1. **НЕ відкривайте публічний issue**
2. Відкрийте [GitHub Security Advisory](https://github.com/AgriciDaniel/claude-seo/security/advisories/new) у цьому репозиторії
3. Або зв'яжіться з мейнтейнером напряму

## Терміни реагування

- **Підтвердження**: Протягом 72 годин після звіту
- **Оновлення статусу**: Протягом 7 днів з початковою оцінкою
- **Вирішення**: Ми прагнемо випустити виправлення протягом 30 днів для підтверджених вразливостей

## Підтримувані версії

Оновлення безпеки отримує лише остання версія.

## Практики безпеки

- Жодні облікові дані чи API-ключі не зберігаються в цьому репозиторії
- Скрипти встановлення записують лише в каталоги рівня користувача (`~/.claude/`)
- Залежності Python встановлюються в ізольовані віртуальні середовища
